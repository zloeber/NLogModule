## Installation
NLogModule is available on the [PowerShell Gallery](https://www.powershellgallery.com/packages/NLogModule/).

To Inspect:
```powershell
Save-Module -Name NLogModule -Path <path>
```
To install:
```powershell
Install-Module -Name NLogModule -Scope CurrentUser
```
